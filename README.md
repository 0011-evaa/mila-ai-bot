# Mila AI — Language Teacher Bot

Telegram AI-викладач чеської та англійської для україномовної учениці.

## Команди

- `/start` — почати
- `/status` — подивитися профіль
- `/reset` — очистити пам'ять

## Локальний запуск

```bash
pip3 install -r requirements.txt
python3 bot.py
```

## .env

Створи файл `.env`:

```env
TELEGRAM_BOT_TOKEN=...
OPENAI_API_KEY=...
```
